# SolarShield AI

An AI-powered Solar Flare Prediction System built for the ISRO Hackathon.

## Features

- Solar flare detection
- Time-series forecasting
- Interactive dashboard
- Explainable AI
- Real-time prediction

## Tech Stack

- Python
- PyTorch
- XGBoost
- Streamlit
- FastAPI
- Astropy
- SunPy
